What Is Code Poetry?

Code poetry has no formal definition. However, most people seem to agree that it is poetry written either in computer code or by computer code. Notable examples range from Black Perl by Larry Wall to The Deletionist by Amaranth Borsuk, Jesper Juul and Nick Montfort. Visit the resources tab on the Stanford Code Poetry Slam page for more general information about code poetry.
